# Lua Minesweeper - Lua Platform

The Lua platform is a box standard boring Lua application with no additional dependencies.
It simply queries you on the next move and reprints the board in plain text. Please use a monospace font in your terminal!

## Commands

| Command       | Action                                          |
| ------------- | ----------------------------------------------- |
| flag 'c' 'r'  | Toggle the flag in position column (c) row (r)  |
| sweep 'c' 'r' | "Click" the cell in position column (c) row (r) |
